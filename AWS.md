    - Type d'instance:
        - usage général
        - calcul optimisé
        - mémoire optimisée
        - calcul accélé
        - stockage optimisé

    - Type de tarification:
        - à la demande
        - "Savings Plan Amazon EC2"
        - instances réservées
        - instance Stot
        - Hôtes dédiés

    - Type de mise à l'échelle (Auto Scaling Amazon EC2):
        - dynamique
        - prédictive

    - Load Balancing -> Elastic Load Balancing
